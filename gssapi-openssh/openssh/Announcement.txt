GSI-OpenSSH 4.4, based on OpenSSH 5.1p1, is now available from:

http://grid.ncsa.uiuc.edu/ssh/download.html

For installation instructions, please see:

http://grid.ncsa.uiuc.edu/ssh/install.html

We appreciate your feedback and help in testing this version.
Please use Bugzilla to report GSI-OpenSSH bugs:

http://bugzilla.globus.org/globus/enter_bug.cgi?product=GSI-OpenSSH

GSI-OpenSSH 4.4 Major changes

   New features

      * Updated to OpenSSH 5.1p1.

      * Updated to HPN13v5.

      * Added server-side GSSAPIDelegateCredentials option:
        http://bugzilla.ncsa.uiuc.edu/show_bug.cgi?id=376

   Bugs fixed

      * Server fails to close connection with
        UsePrivilegeSeparation=yes:
        http://bugzilla.globus.org/bugzilla/show_bug.cgi?id=6236

GSI-OpenSSH 4.0, based on OpenSSH 4.6p1, is now available from:

  <http://grid.ncsa.uiuc.edu/ssh/download.html>

For installation instructions, please see:

  <http://grid.ncsa.uiuc.edu/ssh/install.html>

We appreciate your feedback and help in testing this version.
Please use our bugzilla system to report GSI-OpenSSH bugs:

  <http://bugzilla.ncsa.uiuc.edu/enter_bug.cgi?product=GSI-enabled%20OpenSSH>

GSI-OpenSSH 4.0 Major changes

   New features

      * Upgraded to OpenSSH 4.6p1.

      * Upgraded to HPN12v17 patch from
        <http://www.psc.edu/networking/projects/hpn-ssh/>.

      * Fixed Tru64 build failure
        (http://bugzilla.globus.org/bugzilla/show_bug.cgi?id=5142).

   Bugs fixed

      * None.

GSI-OpenSSH 5.2, based on OpenSSH 5.6p1, is now available from:

  http://grid.ncsa.uiuc.edu/ssh/download.html

For installation instructions, please see:

  http://grid.ncsa.uiuc.edu/ssh/install.html

We appreciate your feedback and help in testing this version.
Please use Bugzilla to report GSI-OpenSSH bugs:

  http://bugzilla.globus.org/globus/enter_bug.cgi?product=GSI-OpenSSH

Major changes:

  * Update to OpenSSH 5.6p1

  * Add GSSAPIServerIdentity option
    https://bugzilla.mcs.anl.gov/globus/show_bug.cgi?id=7108

  * Make all Globus Usage library errors non-fatal
    https://bugzilla.mcs.anl.gov/globus/show_bug.cgi?id=7111

  * Updates for upcoming GT 5.2 threading model
    https://bugzilla.mcs.anl.gov/globus/show_bug.cgi?id=7081
